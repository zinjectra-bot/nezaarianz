Nezaarianz — Premium Batch Website
====================================

Files:
  index.html
  styles.css
  app.js

Images:
  The site references image paths starting with /a/... (as provided).
  When hosting locally or elsewhere, replace those URLs in index.html
  with your own image paths (e.g. images/group1.jpg).
  If an image fails to load, an elegant gradient fallback is shown.

To run:
  Just open index.html in a browser, or serve the folder with any
  static server (e.g. `npx serve .`).
